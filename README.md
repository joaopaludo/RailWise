# RailWise
Sistema para gerenciamento de linhas ferroviárias

<br>

### Funções
1. Controle de tráfego
2. Agendar e planejar rotas
3. Gestão de vagões e locomotivas
4. Gestão de carga
5. Gestão de manutenção
6. Gestão de funcionários
7. Gestão de problemas
